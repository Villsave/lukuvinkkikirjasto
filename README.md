## Definition of Done
* User Storyjen hyväksymiskriteerit testattu Cucumberilla
* Toteutetun koodin testikattavuus 70%
* Toteutettu koodi menee läpi CI-palvelusta
* Toteutettu koodi Checkstylen mukainen
